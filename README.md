# Modern Hand Tracking AR UI

[![Python](https://img.shields.io/badge/Python-3.11-blue?logo=python&logoColor=white)](https://www.python.org/)
[![OpenCV](https://img.shields.io/badge/OpenCV-4.8-orange?logo=opencv&logoColor=white)](https://opencv.org/)
[![MediaPipe](https://img.shields.io/badge/MediaPipe-0.10-purple?logo=google&logoColor=white)](https://google.github.io/mediapipe/)

A **modern, real-time hand tracking AR interface** built with Python, OpenCV, and MediaPipe.  
Supports multiple gestures with intuitive, interactive visual feedback.

---

## Features

- **Open Hand Gesture**: Full AR interface with rotating hexagons, particle rings, and tech lines.  
- **Pinch Gesture**: Interactive control with dynamic arcs and pulsing glow effects.  
- **Fist Gesture**: Locked state visualization with central pulse and rotating hexagons.  
- **Status Panel**: Displays gesture type, FPS, and timestamp.  
- **Custom UI Effects**: Gradient circles, particle rings, HUD corners, and neon glow visuals.
